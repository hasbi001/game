## Setup
1. import db pada folder db dengan nama game
2. install redis
3. git clone https://github.com/hasbi001/game.git
4. buat file .env 
5. copy paster isi folder .env.example ke file .env
6. compoer install
7. php artisan key:generate
7. php artisan serve
8. buka postman kemudian import collection api di folder postman
9. sesuaikan port di url api dengan yang ada di command setelah jalankan langkah ke 4
